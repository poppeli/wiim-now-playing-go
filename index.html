Wiim Now Playing

Lightweight Now Playing display for Wiim & Linkplay audio devices

⬇ Download for Windows

Windows 10/11  ·  ~15–20 MB  ·  No installation required

Features

  • Automatic device discovery via SSDP/UPnP
  • Album art with fallback to iTunes artwork
  • Playback controls (play/pause, previous, next)
  • Device switching for multiple Wiim devices
  • Fullscreen mode with multiple view layouts
  • Dark theme UI
  • Single .exe — no installation, no dependencies

Requirements

Windows 10/11  ·  Wiim device on the same local network
