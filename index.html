Wiim Now Playing

Lightweight Now Playing display for Wiim & Linkplay audio devices, written in Go

⬇ Download for Windows

Windows 10/11  ·  ~15–20 MB  ·  No installation required

Features

  • Automatic device discovery via SSDP/UPnP
  • Album art with fallback to iTunes artwork
  • Playback controls (play/pause, previous, next)
  • Device switching for multiple Wiim devices
  • Fullscreen mode with multiple view layouts
  • Dark theme UI
  • Single .exe — no installation, no dependencies

Keyboard Shortcuts

Playback (works in all modes)
SpacePlay / Pause
RightNext track
LeftPrevious track
Fullscreen
F11Toggle fullscreen
ESCExit fullscreen
1Full view — title, artist, album, progress
2Title + progress
3Art + progress
4Art only (maximum size)
5Art + corner text (title, artist, album)
9Help screen

Fullscreen view mode is remembered between sessions.
